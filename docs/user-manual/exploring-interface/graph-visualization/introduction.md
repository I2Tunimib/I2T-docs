# Introduction
The Graph Visualization allows you to explore the semantic structure of a dataset as an interactive graph.
In this view mode, **nodes** represent table columns, while **links** illustrate the relationships and mappings between them.

The interface is divided into two main sections:
- **Graph Area**: An interactive space to navigate nodes and links. Selecting a graph element, it displays its specific 
  data. _(Learn more about [Graph Area](./graph-area.md))_
- **Sidebar**: Provides detailed information about the overall graph or any currently selected element.
  _(Learn more about [Sidebar](./sidebar.md))_
