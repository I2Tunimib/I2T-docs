# Introduction
The Toolbar provides quick access to global table actions and management features. It allows you to efficiently perform 
various actions that affect the entire table. Starting from the left, you can:

![Toolbar image](/img/global-actions.png)

- **Switch** between different table views: Tabular, Raw JSON and Graph. _(Learn more about [Visualization](./visualization.md))_
- Perform a **compliance check** on the table. _(Learn more about [Compliance Check](./compliance.md))_
- Run an **automatic annotation** for the whole table. _(Learn more about [Automatic Annotation](./automatic-annotation.md))_
- **Export** the table in different formats: JSON, CSV, RDF, Python pipeline and Jupyter notebook pipeline. _(Learn more about [Export](./export.md))_
- **Save** changes to the server. 
- Open the **settings** panel. 
- Open the built-in SemT-X **tutorial**.
