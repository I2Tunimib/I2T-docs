# Generative AI
Generative AI integration allows you to leverage Large Language Models (LLMs) to perform complex semantic operations
that traditional services might struggle with. By using natural language prompts, you can transform content, improve
reconciliation, and extract new data from knowledge bases.

To access these features, click the `Gen AI` button in the Sub-Toolbar. You can then choose from three main types of AI-driven operations:
- **Modification**: Transform, clean, or format existing cell values. 
- **Reconciliation**: Link mentions to entities using AI-driven matching.
- **Extension**: Generate entirely new columns by extracting or inferring information.
_(Learn more about [Generative AI Services](../discover-services/generative-ai))_
