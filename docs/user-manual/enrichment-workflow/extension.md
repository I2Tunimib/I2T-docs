# Extension
Extension is the process of retrieving additional information from external knowledge bases (such as WikiData or OpenMeteo)
using columns that have already been reconciled. This step allows you to enrich your local dataset with new attributes
and properties, transforming simple labels into structured data.

Simply select the reconciled column, click the `Extend` button in the Toolbar, and choose from the available extension
services. _(Learn more about [Extenders](../introduction))_
